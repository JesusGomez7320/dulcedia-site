Dulce Día Cine — Package v3 (Parallax Hero + Newspaper Layout)
=================================================================
Date: 2025-08-11

Deploy steps:
1) Unzip this folder.
2) Put your hero clip at: assets/hero.mp4   (optional: assets/hero.webm)
   - MP4, 10–20 seconds, muted, ~1080p; keep under ~15–20MB.
3) Add poster still at: assets/images/hero-poster.jpg
4) Replace sample images in assets/images/ as needed.
5) Commit & push → Netlify redeploys.

If the live site shows raw '#' or '##' instead of styled headings:
- Your index might be Markdown or missing the <style> block.
- Replace the entire file with this index.html and push.
